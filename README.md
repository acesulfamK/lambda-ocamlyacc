# lambda-ocamlyacc

## 再現する規則

ラムダ式の再帰的定義
1. 全ての変数はラムダ式
2. M, Nがラムダ式ならば、(M N)はラムダ式
3. Mがラムダ式で、xが変数ならば、@x.Mはラムダ式

ラムダ式の縮約規則
1. @x.@y.M は@xy.Mと表せる
2. M, N, P がラムダ式のとき、 ((M N) P) は (M N P) と表せる
3. 数式中の一番外側の括弧は省略できる。
4. ラムダ抽象@.の中において、一番外側の括弧は省略できる。
